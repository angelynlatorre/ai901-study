# AI-901 Study Journey

A self-paced 4-week study guide for the Microsoft **AI-901: Azure AI Fundamentals** certification (refreshed exam, replacing AI-900 on June 30, 2026).

## What's inside

- **28-day learning path** — ~30 minutes per day, mapped to the official Microsoft Learn course
- **Per-day detail cards** — key concepts, exam-trap reminders, code snippets you should recognize, mandatory reading links, and a self-quiz
- **Progress tracking** — your completed days are saved locally in your browser
- **Mock exam day** built into Week 4

## How to use

1. Open the published page (or `index.html` locally)
2. Click any day to open the full study card
3. Read, hit the reading link, do the self-quiz from memory
4. Click **Mark complete** — your progress persists across sessions
5. Aim for ≥80% on every Microsoft Learn knowledge check before moving on

## Topics covered

- Week 1 — AI workloads, ML fundamentals, computer vision & NLP basics, document intelligence, generative AI, **Responsible AI principles**
- Week 2 — Generative AI, prompt engineering, AI agents, NLP services, speech services
- Week 3 — Computer vision deep-dive, vision transformers & image generation, Azure Content Understanding, hands-on labs in Microsoft Foundry
- Week 4 — Vision lab, extraction lab, agent lab, gap review, mock exam, exam booking

## Local use

Just open `index.html` in any modern browser. Everything runs client-side — no server, no build step, no dependencies.
